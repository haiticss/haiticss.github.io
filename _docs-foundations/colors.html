foundations > COLORS

Define de colors that will be used in the layers.

Are al divided in different groups and defined in separated files. Each group is set in a map, which is useful to create the different color utilities.

Main color file

This file contains a map with all the groups maps. This code structure allows a multi-level loop.

{% highlight scss %} {% include haiticss/src/foundations/_foundations__colors.scss %}{% endhighlight %}

Brand colors

The main brand colors, divided in primary, secondary and accent with by default. Each color has 5 shades.

This structure could be changed or extended to accomplish any project specifications.

{% highlight scss %} {% include haiticss/src/foundations/_foundations__colors-brand.scss %}{% endhighlight %}

Neutral colors

The neutral colors are useful to use in all projects. Are based in different shades of grey, that could be change under the project needs.

White and black are also included,

{% highlight scss %} {% include haiticss/src/foundations/_foundations__colors-neutral.scss %}{% endhighlight %}

State colors

The colors of all possible states: success, info, danger and warning. Each colors has also 5 shades defined.

{% highlight scss %} {% include haiticss/src/foundations/_foundations__colors-state.scss %}{% endhighlight %}

Vendor colors

The vendor colors are the colors defined by 3rd party elements, like for example social networks. Can be defined with some shades too.
