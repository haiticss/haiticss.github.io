foundations > SHADOWS

Define the box shadows, without the colors.

Values should be expressed in PX. If you need a relative unit based on that values, use the core functions on its context.

There are 3 different by default, that could be augmented or modified to satisfy any project needs:

  • Discrete
  • Elevated
  • Sunken
