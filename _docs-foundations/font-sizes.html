foundations > FONT-SIZES

Defines the font sizes used in the other layers, using the shirt sizes nomenclature.

The standard font size is m, which is defined by the variable $s-core__font-size.

All the values should be expressed in px. If you need a relative unit based values, you can use the core functions in the context.

A safe checking is included to ensure this unit specification and the numeric values.
