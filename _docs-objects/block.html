Objects > BLOCK

Stacked content object. A simple abstraction to cover a very commonly occurring design pattern.

Header, body and footer elements are optional.

{% highlight html %}
{% endhighlight %}

Space modifiers

The block space modifiers defines the vertical distance between header, body and footer. The object space by default is medium.

{% highlight html %}
{% endhighlight %}

Alignment modifiers

The block alignment modifiers defines the horizontal text alignment. The object alignment by default is center.
